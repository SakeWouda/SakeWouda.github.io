Geheugen

RAM

Het Random Access Memory is het werkgeheugen van een systeem. De RAM bestaat uit geheugencellen die tijdelijk informatie opslaan. Door een toevoer van stroom met een hoge frequentie wordt het geheugen onthouden. Het RAM wordt gebruikt om snel informatie aan de CPU te leveren.

ROM

Het Read Only Memory blijft altijd opgeslagen in de computer. Deze zorgt ervoor dat alle noodzakelijk gegevens toegankelijk zijn voor de CPU. Het BIOS (Basic input / Output system) wat ook in het ROM staat voert een paar test uit voordat de computer wordt opgestart.

Permanent geheugen

De twee varianten van permanent geheugen zijn elektrische en magnetische opslag.

Het elektrische opslag, ook wel flash geheugen, werkt hetzelfde als het werkgeheugen, maar onthoudt net als het ROM altijd de data. Ook als de elektrische spanning verdwijnt. Het heet flash geheugen, omdat in een flits alle gegevens kunnen worden verwijdert en toegevoegd. Een voorbeeld hiervan is de usb-stick

De magnetische opslag bevat luchtdicht behuizing met daarin een aantal dunne schijven van aluminium of glas. Deze zijn bedekt met een laag magnetisch materiaal en draaien met een grote snelheid rond. Elke schrijfkop kan data in de laag van een plaat aanbrengen in de vorm van een serie van enen en nullen. De opslagcapaciteit wordt bepaald door het aantal schijven en de samenstelling van de magnetische laag.
