Processor

De processor die ook wel CPU wordt genoemd, geeft alle instructies aan het systeem. Het is het brein van je computer.

In sommige systemen bevinden zich meerdere CPU’s. Dit worden multiprocessor systems genoemd. Soms is er één processor, maar meerdere kernen. Maar deze zijn alleen sneller als er meerdere taken tegelijk moeten worden uitgevoerd.

De CPU voert steeds dezelfde cyclus uit. De FDE-cyclus, Fetch-Decode-execute. Hoe goed de cpu is, hangt af van hoe snel de cyclus kan worden uitgevoerd.

De CPU bestaat uit het besturingsorgaan en het rekenorgaan. De taken hiervan zijn:

-Het opslaan van resultaten

-Berekeningen uitvoeren

-Ophalen van gegevens

-Regelen van de output

Kloksnelheid & verwerkingssnelheid

De kloksnelheid is hoe vaak de CPU een puls geeft. De bijpassende eenheid is Hz. De verwerkingssnelheid heeft ook te maken met de snelheid van CPU. Deze houdt in hoe vaak de FDE-cyclus kan worden uitgevoerd. Deze ligt altijd lager dan de kloksnelheid.
